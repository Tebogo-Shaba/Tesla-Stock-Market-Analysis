# Tesla Stock Market Analysis

## Project Overview
This project analyzes Tesla's stock prices from 2010–2025 using Python.  
Focus is on time series trends, volatility, and data storytelling.

## Features
- Time series visualization of closing prices  
- Moving averages to identify trends (20-day, 50-day)  
- Daily returns to measure volatility  
- Observations and insights documented in Markdown

## Technologies
- Python (Pandas, Matplotlib)  
- Kaggle Notebook

## Usage
1. Open the Kaggle notebook: [https://www.kaggle.com/code/tebogoshaba/stock-market-analysis]  
2. Attach dataset: `tesla-stock-price-historical-dataset-2010-2025`  
3. Run all cells to reproduce analysis

## Insights
- Tesla shows strong growth post-2020  
- Periods of high volatility coincide with market events  
- Moving averages indicate trend reversals
